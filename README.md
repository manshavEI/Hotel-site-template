# 🏨 Hotel Website Template

A modern, responsive hotel website template built using HTML, CSS, and JavaScript. The project focuses on creating an elegant user experience with a clean interface, smooth interactions, and mobile-friendly design.

## ✨ Features

* Responsive design for all screen sizes
* Modern and clean UI
* Hero section with call-to-action
* About section
* Rooms & Services showcase
* Gallery section
* Contact section
* Smooth scrolling navigation
* Interactive UI elements
* Cross-browser compatibility

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
